# CairnMap Presets

This directory stores native, low-priority configuration presets. The OpenRIAMap RIA package should assemble these presets, then apply project-level overrides on top.

Preset groups:

- `core-structures`: generic point/line/surface structures.
- `building`: building and floor structures.
- `rail`: railway, station, platform, and station-building structures.
- `road`: road structures.
- `teleport`: teleport points.
- `warp`: warp points.
- `trade`: trade points.
